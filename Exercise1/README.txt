ΣΥΝΑΡΤΗΣΕΙΣ:

int HP_CreateFile(char *fileName) : Στην αρχή ελέγχει αν ο φάκελος με όνομα fileName υπάρχει ήδη και αν όχι , τον 
δημιουργεί. Αφότου τον δημιουργήσει , δεσμέυεται χώρος στη μνήμη για καινούριο μπλοκ στο οποίο αποθηκεύονται τα 
μεταδεδομένα του αρχείου και τίποτα άλλο μιας και δεν χρειάζεται. Χρειάστηκε να χρησιμοποιήσουμε την BF_OpenFile
έτσι ώστε να μπορέσουμε να δεσμεύσουμε χώρο για το πρώτο μπλοκ που κρατάει τα μεταδεδομένα.
---------------------------------------------------------------------------------------------------------------------

HP_info *HP_OpenFile(char *fileName, int *file_desc) : Στην αρχή δεσμεύεται χώρος για την δομή HP_info που αποθηκέυει 
τα μεταδεδομένα του αρχείου και έπειτα διαβάζουμε τα μεταδεδομένα του αρχείου μέσω της συνάρτησης fread() και στη 
συνέχεια χρησιμοποιείται ξανά η BF_OpenFile() έτσι ώστε να δωθεί το τελικό αναγνωριστικό του αρχείου και τελικά 
επιστρέφεται η δομή hp_info.
---------------------------------------------------------------------------------------------------------------------

int HP_CloseFile(int file_desc, HP_info *hp_info): Η συνάρτηση αυτή ελευθερώνει πρώτα τον χώρο που καταλαμβάνει 
η δομή hp_info που δίνεται ως όρισμα και έπειτα χρησιμοποιείται η συνάρτηση BF_CloseFile για να κλείσει το αρχείο που 
θέλουμε να κλείσουμε και να αποδεσμεύσουμε τη μνήμη του.
---------------------------------------------------------------------------------------------------------------------

int HP_InsertEntry(int file_desc, HP_info *hp_info, Record record): Η συνάρτηση αυτή χωρίζεται σε 3 περιπτώσεις.
1η περίπτωση είναι να μην υπάρχει κανένα μπλοκ εγγραφών στο αρχείο, όπου αυτό που πρέπει να γίνει είναι να δεσμέυεται 
	η μνήμη για ένα μπλοκ και να εισάγουμε τα μεταδεδομένα του μπλοκ στη αρχή του. Έπειτα, μετά τα μεταδεδομένα,
	πρέπει να εισάγουμε την εγγραφή που μας δίνεται ως όρισμα και να κάνουμε το μπλοκ dirty και unpin.
2η περίπτωση είναι να κάνουμε προσθήκη σε ήδη υπάρχων μπλοκ, όπου αυτό που πρέπει να γίνεται είναι να ανοίγουμε το 
	τελευταίο μπλοκ του αρχείου , να προσπερνάμε τα μεταδεδομένα του που είναι στη αρχή και έπειτα εφόσον χωραέι 
	άλλη εγγραφή, να μπει στο τέλος του μπλοκ αυτού και να μετατραπεί το μπλοκ σε dirty και να κάνουμε unpin.
3η περίπτωση είναι να μην χωράει το προηγούμενο μπλοκ όπου θα πρέπει να δημιουργηθεί καινούριο μπλοκ , να φτιάξουμε 
	μεταδεδομένα για αυτό το μπλοκ και να τα εισάγουμε στην αρχή του και έπειτα να εισάγουμε την καινούρια εγγραφή 
	μετά τα μεταδεδομένα και να θέσουμε το προηγούμενο μπλοκ να δείχνει στο καινούριο. Τέλος το θέτουμε ως dirty 
	και το κάνουμε unpin(το καινούριο μπλοκ).
----------------------------------------------------------------------------------------------------------------------

int HP_GetAllEntries(int file_desc, HP_info *hp_info, int value): Η συνάρτηση αυτή δημιουργεί έναν δείκτη τύπου μπλοκ,
έναν δείκτη μεταδεδομένων μπλοκ, έναν δείκτη void και έναν δείκτη σε εγγραφές. Έπειτα ξεκινάει βρόχος από το 2ο μπλοκ 
του αρχείου (το 1ο περιέχει μεταδεδομένα μόνο), έως το τελευταίο μπλοκ του αρχείου και αποθηκεύουμε τις πληροφορίες του 
κάθε μπλοκ στους δείκτες που δημιουργήθηκαν στην αρχή, έτσι ώστε να έχουμε πρόσβαση και στις εγγραφές του κάθε μπλοκ 
για να μπορέσουμε να δούμε ποιες από αυτές έχουν το id τους ίσο με το value που περνιέται ως όρισμα.
Αυτό γίνεται με εσωτερικό βρόχο σε κάθε μπλοκ για να δούμε όλες τις εγγραφές του και να εκτυπώσουμε αυτές με το id που 
θέλουμε. Στο τέλος κάθε βρόχου αποθηκεύεται ο αριθμός του μπλοκ στο οποίο βρισκόμαστε έτσι ώστε να επιστραφεί στο τέλος 
πόσα μπλοκ διαβάστηκαν από τη συνάρτηση.
